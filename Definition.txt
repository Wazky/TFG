Propuesta TFG

Titulo Provisional: 
    App web planificador de menús semanales


-Objetivo General:
    Desarrollar una aplicación web que genere menús equilibrados y optimice 
    la compra de alimentos y reduzca el desperdicio alimentario mediante
    recomendaciones de comida.

-Objetivos Específicos:

    ·Usuario pueda registrar sus propias recetas de comida.
    
    ·Implementar un algoritmo de generación automática de menús semanales equilibrados.

    ·Crear sistema de recomendación que sugiera recetas complementarias basadas en ingredientes comunes.

    ·Generar una lista de la compra correspondiente para los menús semanales ¿?

    ·Recetario global sobre el que poder buscar recetas (opciones de búsqueda: por alimento, de temporada, tipo de plato (entrante, ensalada,...)) ¿?


-Funcionalidades:

    ·Módulo de usuarios: registro, login, perfil con preferencias alimentarias (alergias, preferencias alimnetarias).
    ·Gestor de recetas: añadir recetas con ingredientes, cantidades por persoa, categorías, valores nutricionales, pasos de preparación, ...
    ·Generador de menús:    
        -Crea menús semanales equilibrados (proteínas, verduras, hidratos, calorias, ...).
        -Evita repeticiones excesivas en los menús semanales.
    ·Sistema de recomendaciones:
        -Sugerencias de recetas adicionales cuando no hay suficientes.
        -Optimización  basada en reutilizar ingredientes.
    ·Lista de la compra: genera una lista de la compra con los ingredientes para hacer los menús de la semana.


-Tecnologías Arquitectura Base:
	·Backend: JAX-RS con Tomcat.
	·Front: HTML, CSS/Bootstrap, Javascript, JQuery.
	


-Tecnologías Ideas Propuestas:

    ·Frontend:
        React o Angular
        TailwindCSS (Alternativa a Bootstrap/CSS)

    ·Backend (API + Lógica de negocio):
        Django REST Framework (Python) podría integrar algoritmos fácilmente
        Node.js con NestJS 
        Java + Javascript

    ·Base de Datos:
        PostgreSQL (cómodo para datos estrcuturados)
        MySQL

    ·Algoritmos de optimización:
        Pandas/Numpy análisis de nutrientes
        OR-Tools heurísticas propias para el equilibrio nutricionales
